# CodeAlpha1_hangman_game
Hangman Game

A simple Hangman game developed in Python as part of the CodeAlpha Python Programming Internship.

Features

* Random word selection
* Input validation
* Win/Loss detection
* Hangman ASCII art
* User-friendly interface

Technologies Used

* Python

How to Run

1. Open terminal in the project folder.
2. Run:

py hangman.py

3. Start guessing letters and enjoy the game!
